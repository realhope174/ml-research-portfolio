## Projects

### [1. Credit Risk Scoring with XGBoost and SHAP](./01_credit_risk_xgboost)
End-to-end credit default prediction model using XGBoost with SHAP-based interpretability. 
Explores how individual predictions can be explained and how model decisions vary across 
the feature space — directly relevant to fairness and explainability in financial ML systems.

### [2. Counterfactual Recourse for Credit Decisions](./02_counterfactual_recourse)
Implements counterfactual explanation methods to identify minimal actionable changes 
that would flip a model's decision for a rejected applicant. Connects to causal recourse 
frameworks and the model multiplicity problem in high-stakes decision systems.

### [3. Time Series Forecasting: Exchange Rate Modelling](./03_time_series_forecasting)
ARIMA-based forecasting model built on ten years of GBP/GHS monthly exchange rate data. 
Includes stationarity testing, ACF/PACF diagnostics, residual analysis, and 
structured forecast outputs with confidence intervals.

### [4. Monte Carlo Simulation of the 2D Ising Model](./04_monte_carlo_simulation)
Metropolis-Hastings Monte Carlo simulation of spin system behaviour across temperature 
conditions. Demonstrates expertise in probabilistic simulation and stochastic modelling 
under uncertainty.
